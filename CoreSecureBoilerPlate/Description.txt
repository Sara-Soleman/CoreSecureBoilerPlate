Here, we will build an ultra-fast and lightweight API using Minimal APIs in .NET 10 to connect endpoints directly to commands and queries via the MediatR library.

In .NET 10, Minimal APIs have become the standard for building modern microservices and APIs because they eliminate bulky controller files, 
allow for direct injection of handles and ISenders into variables with very clean and concise code, and support Native AOT for extremely fast execution.
